# Isharpe API

Projeto pra ranquear todos os fundos de investimento da CVM por sharpe.

## Configuração

- Node.JS
- Express
- Nodemon (para autoreload no desenvolvimento)
- Sucrase (permite usar sintaxe ES6+ e TS)
- PostgreSQL
- ORM Sequelize
- Eslint, Prettier e EditorConfig
- JWT e Bcrypt (auth)
- Yup (Validações)
